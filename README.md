# xin-codemirror

[github](https://github.com/tonioloewald/xin-codemirror/) | [live demo](https://tonioloewald.github.io/xin-codemirror/) | [npm](https://www.npmjs.com/package/xin-codemirror)

**blueprint src url** `https://tonioloewald.github.io/xin-codemirror/dist/blueprint.js`

This is a blueprint for a [codemirror 6]() based code editor.

The blueprint is statically bundled with with support for html, css, typescript, javascript, tsx, jsx, and json. If you want to change the available languages or specify a different version of codemirror, feel free to fork this.
